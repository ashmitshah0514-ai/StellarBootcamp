# 🎯 Earn Quests — Soroban Smart Contract

## 📌 Project Description

**Earn Quests** is a decentralized quest-and-reward system built on the **Stellar Soroban** smart contract platform. It enables creators, platforms, or communities to design engaging tasks (“quests”) that users can complete to earn rewards in a transparent and tamper-proof way.

By leveraging Stellar’s fast and low-cost infrastructure, Earn Quests introduces a scalable way to gamify participation across Web3 and traditional applications.

---

## ⚙️ What it does

The Earn Quests contract powers a simple yet powerful reward engine:

* 🛠️ **Quest Creation** — Admins define quests and assign rewards
* 🎮 **Quest Completion** — Users complete quests and claim rewards
* 💰 **Reward Tracking** — User rewards are stored securely on-chain
* 🔍 **Transparency** — All actions are verifiable on the blockchain

This ensures fairness and eliminates the need for centralized reward management.

## 🧪 How It Works (Concept Flow)

1. Admin creates a quest with a reward value
2. User completes the quest
3. Smart contract verifies and records completion
4. Reward is credited to the user’s on-chain balance
